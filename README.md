# Chennai Food Pattern Calendar 2026

A culturally-aware calendar that predicts **veg / non-veg food patterns** in Chennai based on:

- Tamil calendar (solar months)
- Lunar cycles (Amavasya, Pournami, Ekadashi, Pradosham)
- Religious observances (Hindu, Christian, Muslim)
- Real-world eating habits

---

## 🌐 Live App

👉 https://baijunt.github.io/chennai-food-calendar/

## 📱 Features

### 📅 Full 2026 Calendar
- Month-wise view
- Mobile-friendly scroll (day-by-day on phone)

### 🟢 Food Pattern Indicators
- 🟢 Veg days
- 🔴 Non-veg days
- 🟡 Flexible days

### 🌙 Sacred Day Awareness
Includes:
- Amavasya (New Moon)
- Pournami (Full Moon)
- Ekadashi (Fasting days)
- Pradosham (Evening temple influence)

### 🧠 Smart Insights
- Tooltip explanations for each special day
- Example:
  - *Pradosham → High veg snack demand (5–8 PM)*

### 📍 Tamil Calendar Accuracy
- Real Tamil month boundaries (Aadi, Purattasi, etc.)
- Not based on simple Gregorian assumptions

### 📱 Mobile App Experience
- Scrollable day cards
- Tap to view details
- “Today” auto-scroll + highlight
- Installable as an app (PWA)

## 🧾 How It Works

The calendar combines:

- 📆 Weekday patterns (Tue/Fri veg, Sunday non-veg)
- 🌙 Lunar tithi cycles
- 🛕 Temple-based practices
- ✝️ Christian observances (Good Friday, Easter)
- ☪️ Ramadan & Eid
